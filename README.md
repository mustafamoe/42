## Python 01

Object-oriented garden system exercises covering:

- program entry points with `if __name__ == "__main__"`
- classes, objects, attributes, and methods
- constructors with `__init__`
- encapsulation with protected attributes, getters, and setters
- inheritance, `super()`, and method overriding
- static methods, class methods, and nested classes
